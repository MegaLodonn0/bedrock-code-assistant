🚀 QUICK START

1. Install dependencies:
   $ pip install -r requirements.txt

2. Configure credentials:
   $ cp config.example.json config.json
   $ # Edit config.json with your AWS credentials

3. Run the assistant:
   $ python main.py

4. Try commands:
   > ask What is Python?
   > models
   > exec python --version
   > help
   > exit
